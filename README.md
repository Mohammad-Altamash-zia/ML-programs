# Machine Learning Programs & Assignments

This repository contains a collection of Machine Learning scripts, models, and assignments completed as part of the B.Tech CSE coursework at Invertis University. 

## 📌 Topics & Algorithms Covered
Based on the curriculum, this repository includes practical implementations of:
* **Supervised Learning:** Linear Regression, Logistic Regression
* **Unsupervised Learning:** Clustering
* **Predictive Modeling:** House Price Prediction, Student Marks Prediction
* **Data Preprocessing:** Handling Dummy Variables

## 💻 Technologies Used
* Python
* Scikit-Learn
* Pandas & NumPy
